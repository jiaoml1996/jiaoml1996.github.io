论文笔记《Unified Multi-scale Feature Abstraction for Medical Image Segmentation》

发表于 2020-03-03 | 更新于 2021-05-18 | 分类于 医学图像分割
本文字数： 2.8k | 阅读时长 ≈ 3 分钟

用于医学图像分割的统一多尺度特征提取

DESCRIPTION OF PURPOSE

医学图像自动分割是医学图像分析的重要组成部分，在计算机辅助诊断中起着重要的作用。例如，肝的定位和分割对肝癌的诊断和治疗有很大的帮助。医学图像分割的最新模型是编码器-解码器结构的变体，如全卷积网络(FCN)和U-Net。基于FCN的分割方法的一个主要关注点是通过结合最新的CNN结构(如ResNet和DenseNet)进行网络结构工程。除了探索新的网络结构来有效地提取高层次特征外，在FCN中加入多尺度图像特征提取的结构有助于提高分割任务的性能。在本文中，我们设计了一种新的多尺度网络结构，它采用多尺度输入和专门的卷积路径来有效地结合来自不同尺度的特征，从而更好地利用层次信息。

METHODS

该MIMO-FAN首先利用空间金字塔池获取场景上下文信息，对输入图像进行多尺度分析。在使用共享内核的第一级卷积块之后，可以从这些输入中以不同的比例提取解释整个场景的图像级上下文特征。从这里开始，MIMO-FAN的一个显著特性是，要在一定级别上进行融合的特性都要使用密集跨尺度连接(DCCs)通过相同数量的卷积层，这有助于保持层次结构，以便更好地进行抽象。每个DCC模块对卷积块和相同深度的不同尺度特征之间的密集连接使用Res连接。与经典的基于U-net只在卷积深度增加时尺度减小的方法不同，而MIMO-FAN在每个深度都具有多尺度特征，因此可以充分集成全局和局部上下文信息来增强提取的特征。此外，受深度监督工作的启发，我们进一步将深度金字塔监督(deep pyramid supervision, DPS)引入解码端，生成和监督不同尺度的输出，这有助于缓解梯度消失问题，在不同尺度下生成良好的分割掩码。最后，将两个最大的概率图融合在一起，通过尺度融合(SF)实现更可靠的分割。DCC和DPS模块的详细信息如下所示。

Dense Cross-scale Connections (DCCs)

研究表明，全局特征提取和上下文集成有利于语义分割。我们不是在卷积网络的最后阶段提取多尺度特征，而是从网络一开始就获取多尺度特征，以保留输入图像的上下文信息，并在整个网络中利用不同尺度的特征。在某种程度上，规模越小，特征可能包含的全局上下文信息就越多。因此，为了有效地增强特征表示能力，我们开发了具有新的跳跃连接的DCC块，将不同尺度的特征映射融合在同一层次上，如图1(B)所示。我们在同一层次上添加了不同尺度特征之间的紧密联系。通过DCCs将不同尺度的特征进行融合和重用，使特征更具代表性，包含更多层次信息。在编码器部分，MIMO-FAN使用自顶向下的连接来组合多尺度的feature map，而在解码器部分使用自底向上的顺序来逐步解码高层的feature。

Deep Pyramid Supervision (DPS)

为了在小尺度和深度层次上实施有效的特征提取，我们提出了深度金字塔监督(deep pyramid supervision, DPS)来监督不同尺度上的输出。为了处理输出大小的变化，我们对ground truth segmentation进行空间金字塔池操作，生成所有输出尺度的标签。在相同的训练规模下，利用输出和地面真值分割计算训练损失。在我们的工作中，采用加权交叉熵作为损失函数，定义为: \[ L=-\frac{1}{S}\sum^{S}_{s=1}\sum^{1}_{c=0}w^{c}_{i,s}y^{c}_{i,s}\log{p^{c}_{i,s}} \] 其中\(p^{c}_{i,s}\)表示像素i在规模s的情况下的类别c，\(y^{c}_{i,s}\)是规模s下的ground truth，\(N_s\)表示在规模s下像素的个数，\(w^{c}_{i}\)表示不同类别的权值参数。根据经验，我们将背景的权重设为0.2，肝脏的权重设置为1.2。在我们的工作中，尺度规模设置为5，如图Fig. 1(A)

RESULTS

我们在由131个训练数据和70个测试数据组成的LiTS(肝肿瘤分割挑战)数据集上对我们的模型进行了广泛的评估。数据来自不同医院，CT扫描层内分辨率为0.45mm - 6mm，层间分辨率为0.6mm - 1.0mm。每个切片的大小为512×512像素。为了加快模型训练的速度，我们将轴向切片大小调整为256×256像素，其中边界信息仍然保存得很好。采用五次交叉验证来评估模型在挑战训练数据集上的性能。在准备向挑战赛提交测试结果时，我们使用了多数投票的方式将五种模型的输出结果结合起来，得到最终的分割结果。一旦论文被接受，我们的实现代码将是开源的。

Comparison with other methods

目前最先进的肝脏CT图像分割方法大多需要两个步骤来完成，首先是粗分割来定位肝脏，然后是精细分割来获得最终的分割结果。许多工作将二维和三维特征结合在一起，以提高分割性能。然而，这些方法在计算上是很昂贵的。例如，Li等人的方法需要21个小时来训练2D DenseUNet，然后用两个Titan Xp gpu再花9个小时来完善H-DenseUNet。通过对比，我们提出的方法可以在3小时内在一个Titan Xp GPU上完成一个模型的训练。此外，我们的2D网络将肝脏一步分割，可以获得非常有竞争力的性能，与顶级执行方法相比，DICE的差异小于0.5%，如表1所示。

Ablation study

我们进一步比较了我们提出的MIMO-FAN与其他几个经典网络，包括U-Net、ResU-Net和DenseU-Net，以证明DCC和DPS的有效性。部分算例结果如图2所示。U-Net、ResU-Net和MIMO-FAN都是具有相同数量过滤器的19层网络。我们在相同的环境中从零开始训练所有这些2D网络。五倍交叉验证结果如表2所示。经t检验，MIMO-FAN显著优于U-Net、ResU-Net和DenseU-Net, p值分别为0.004、0.025和0.002。

NEW OR BREAKTHROUGH WORK TO BE PRESENTED

据我们所知，所提出的MIMO-FAN是第一个将多尺度输入和多尺度输出特性集成到一个网络中的高效抽象网络架构。我们在MICCAI 2017肝脏肿瘤分割(LiTS)挑战数据集上对该方法进行了广泛的评估，并展示了良好的性能。

CONCLUSION

在这篇文章中，我们提出了一种统一的多尺度特征提取的网络结构，该结构在不同深度上以分层的方式处理多尺度特征，用于图像分割。与其他三维网络相比，二维网络在肝脏CT图像的单步分割中具有很强的竞争力。该方法也可以应用于其他的分割任务，并将在未来的工作中进行测试。
