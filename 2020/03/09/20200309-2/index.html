论文笔记《Semi-Supervised Semantic Image Segmentation using Dual Discriminator Adversarial Networks》

发表于 2020-03-09 | 更新于 2021-05-18 | 分类于 半监督 ， 语义分割

利用双鉴别器对抗网络进行半监督语义图像分割

Abstract

基于卷积神经网络的语义图像分割方法需要大量的像素级训练数据，但标记过程耗时费力。在本文中，我们提出了一种半监督语义分割方法，它可以利用模型训练中的未标记数据来减轻标记任务。提出了一种由生成网络和双鉴别网络组成的新型GAN网络结构，并将标准的多类交叉熵损失与竞争损失耦合起来对整个网络进行训练。为了进一步提高目标边界的定位能力，在生成网络中增加了一个自关注层，对图像中的远程依赖关系进行建模，并增加了一个跳跃层，将深层与高度抽象的信息和浅层与详细的外观信息结合起来。该双鉴别器网络包括一个全卷积鉴别器和一个典型的GAN鉴别器，使输入图像在像素级和图像级都能被识别。在半监督语义分割中，首先利用图像级鉴别器对未标记图像的预测分割结果进行筛选，然后利用像素级鉴别器生成未标记图像的可信区域，提供额外的监督信号。在PASCAL VOC 2012数据集上的大量实验表明，我们的方法在精度上优于现有的半监督语义图像分割方法。

关键词:语义分割，半监督学习，生成对抗网络。

Introduction

语义图像分割是计算机视觉图像理解的重要组成部分。它的目的是为输入图像中的每个像素分配一个语义标签，并输出具有像素级语义标注的图像。这个任务被广泛应用于许多领域，例如自动驾驶和图像编辑[1]。近年来，基于深度卷积神经网络(CNN)尤其是全卷积网络(FCN)[2]的方法极大地促进了语义分割的发展。已经有许多算法被提出，其中一些已经取得了很好的性能[2,3]。然而，基于CNN的方法需要大量的带标记的训练数据，并且缺乏像素级的带注释的图像，使得这些方法难以应用于实践。

为了缓解缺少高质量带注释的训练数据的问题，最近提出了许多弱/半监督语义分割方法。弱监督方法经常利用图像级别的标签（例如对象类）来提供辅助监督信号。例如，[4]中的方法使用经过类标记图像训练的多实例学习（MIL）框架来生成潜在特征图，并对这些图进行聚合以约束模型，以将更多权重分配给正确的像素。但是，这些弱监督方法效果不佳，因为它们无法从弱监督信号中恢复详细的对象边界。对于半监督方法，联合使用一些全注释图像和弱标记图像来训练分割网络。例如，Hong等人[5]建立了一个由分类网络和分割网络组成的框架，其中，分类网络作为特征提取器，被训练带有大量带有目标类标签的图像，并进行分割网络使用少量全注释图像进行训练。

自提出以来，生成对抗网络（GAN）[6]引起了广泛的关注。一个典型的GAN模型由一个生成器和一个鉴别器组成，这两个部分在训练过程中扮演minimax游戏。它可以广泛用于许多领域，例如图像生成[7]和语义分割[1，8，9]。 [8]中的方法是将对抗训练应用于语义分割的第一项工作。但是，它不是适用于不同数据集的通用方法，并且与基线方法相比并没有取得实质性进展。作为一种半监督的语义分割方法，Hung等人。 [1]设计了一种全卷积鉴别器，为未标记的数据提供监督信号。但是，由于缺乏图像级鉴别器，该方法无法判断输入概率图是来自分段输出还是来自全局真实性，此外，为未标记数据提供的监控信号不太准确。 Souly等。 [9]采用了不同的观点，将鉴别器视为细分网络，并使用生成的样本来扩展训练数据。由于生成的图像与真实样本的距离不够近，因此它们无法有效地提高分割网络的性能[1]。

在本文中，我们提出了一种带有双重鉴别器的GAN模型，用于半监督语义分割。 双重鉴别器可以在像素级别和图像级别上将生成的概率图与地面真值标签图区分开。 对于未标记的图像，它通过在生成的分割结果中发现可信赖的区域来提供其他监控信号。 分割网络用作生成器，其中利用了两个技巧来恢复详细的对象边界。 第一个是自我注意层，它计算一个位置处的响应，将其作为特征图中所有位置的加权总和，以对远程依存关系进行建模[10、11]。 第二个是跳过层，通过将浅层，精细和详细信息与深层，粗略和高度抽象的信息融合来改善分割细节[2]。

Proposed Method

在这一节，我们将详细介绍算法结构。图1是提出的由segmentation network (G)和discriminator network (D)组成的网络框架。分割网络将\(H\times W\times 3\)的RGB图像作为输入，然后输出\(H\times W\times C\)的分割概率图，其中C是语义类别的个数。鉴别器网络包括两个sub-discriminators,𝐷1是一个典型的鉴别器和𝐷2是一个完全卷积鉴别器，使用分割网络的分割图或者ground truth作为输入，D1输出一个概率值，D2输出一个概率图，也叫做置信图，大小为HxWx1.

Network Architecture

分割网络 分割网络可以是任何为语义分割而设计的网络，如FCN[2]和DeepLab[3]。与[1]类似，我们采用DeepLab-v2[3]框架，以ResNet-101[12]作为我们的基线分割网络(G)。为了使整个模型适合于一个GPU，我们删除了[3]中使用的多尺度输入，以减少内存占用。此外，我们没有使用完全连接条件随机场(CRF)的后处理方法，因此我们的网络是端到端的训练来进行结构化预测。考虑到简化DeepLab将模糊对象边界的定位,我们添加一个self-attention层(10、11)之间的第二个和第三个残块resnet - 101模型远程依赖项,因此分割网络能够获得社区的对象形状信息的对象,而不是当地区域的固定形状[11]。我们还增加了一个跳跃层，将第三个残差块的特征映射直接引入到分类器Atrous Spatial Pyramid Pooling (ASPP)[3]中，因为网络的浅层包含更丰富的语义信息，有助于产生准确、详细的分段。

对于自注意力层，图1中黄色块和图2中展示的那样，我们首先通过前面的隐层\(x\)转换成三个不同的功能空间f, g和h在特征图\(x\)（\(h\times w\times c\)）上实现了1x1卷积，其中\(f(x)=W_{f}x,g(x)=W_{g}x,h(x)=W_{h}x\).

第二步，我们计算了\(x_{i}\)和相同通道所有位置的相似度\(\beta_{i,j}\)。\(\beta_{i,j}\)表示模型在合成第i个位置时对第j个位置的权重: \[ \beta_{i,j}=\frac{exp(s_{ij})}{\sum^{h\times w}_{j=1}exp(s_{ij})},where s_{ij}=f(x_{i})^{T}g(x_{j}), \tag{1} \] 第三步，我们计算注意力层\(y_{i}\)的输出： \[ y_{i}=\sum^{h\times w}_{j=1}\beta_{i,j}h(x_{j}), \tag{2} \] 最后，我们缩放了自注意层的输出，并将其添加回输入隐藏特征图。最终输出由: \[ z_{i}=\gamma y_{i}+x_{i}, \tag{3} \]

鉴别网络 为了适应semi-supervised学习任务的方法,我们设计一个双鉴别器网络(D)与CNN的形式,其中𝐷2是完全卷积鉴别器和𝐷1是一个典型的鉴别器。鉴别器网络由5个卷积层组成，4×4的核尺寸和stride 2，一个池化层和一个全连接层。𝐷1之上的所有层和输出包含一个值的概率代表输入图像来自地面实况标签地图。卷积𝐷2包含5层,然后up-sampling层重新调节输出大小𝐻×𝑊×1。

Training Process and Loss Function

我们使用以下符号表示，\(X_{n}\)表示大小为\(H\times W\times 3\)的RGB图像。\(Y_{n}\)表示大小为\(H\times W\times C\)的ground truth标签图，其中C是类别数，\(S(.)\)表示分割网络，\(S(X_{n})\)表示分割网络对输入图像\(X_{n}\)生成的大小为\(H\times W\times C\)的分割图。\(D_{1}(.)\)表示图像级的鉴别器，\(D_{2}(.)\)表示像素级的鉴别器。

鉴别网络 为了训练双鉴别器网络，我们最小化被\(L_{D_1}\)和\(L_{D_2}\)计算的二值分类交叉熵损失： \[ L_{D}=-(\sum^{H\times W}_{i=1}\log{(1-D_{2}(S(X_n))^{(i)})}+\log{(D_2(Y_n)^{(i)}}))-\lambda (\log{(1-D_1(S(X_n)))}+\log{(D_1(Y_n))}), \tag{4} \] 公式4中的第一项是鉴别器\(D_2\)的交叉熵损失\(L_{D_2}\)，i代表大小为\(H\times W\times 1\)的概率图中的第i个像素。第二项是鉴别器\(D_1\)的交叉熵损失\(L_{D_1}\)，\(\lambda\)是平衡\(L_{D_1}\)和\(L_{D_2}\)之间的超参数。我们只是用有标注数据训练鉴别器网络。

分割网络 为了训练分割网络，我们最小化三项加权和的多任务损失函数\(L_{seg}\)： \[ L_{seg} = L_{mce}+\lambda_{adv}L_{adv}+\lambda_{semi}L_{smei}, \tag{5} \] 第一项\(L_{mce}\)是多类交叉熵损失，他是语义分割模型中标准的损失函数。\(L_{adv}\)是对抗损失，\(L_{semi}\)是半监督损失，\(\lambda_{adv}\)和\(\lambda_{semi}\)是两个超参数。

对于有标注数据\(X_{n}\),他的ground truth是\(Y_{n}\)，生成的概率图是\(S(X_n)\)，多类交叉熵损失： \[ L_{mce}=-\sum^{H\times W}_{i=1}\sum^{C}_{c=1}Y^{i,c}_{n}\log{(S(X_n)^{(i,c)})}, \tag{6} \] 对抗损失由\(D_1\)和\(D_2\)计算： \[ L_{adv}=-\sum^{H\times W}_{i=1}\log{(D_2(S(X_n))^{(i)})}-\lambda \log{(D_1(S(X_n)))}, \tag{7} \] 对于无标注数据\(X_n\)，我们不能直接计算\(L_{mce}\)因为没有相应的ground truth \(Y_n\)，但是\(L_{adv}\)是可用的。因此，在半监督学习框架中，我们使用训练好的双鉴别器来为未标记的数据提供辅助的监督信号。主要的idea是我们首先使用\(D_1\)鉴别生成的分割图。如果无标注数据\(X_n\)的\(D_1(S(X_n))>\lambda_{th}\)，然后\(Y^{'}_{n}=argmax(S(X_n))\)被当作伪标签。通过\(D_2\)的\(D_2(S(X_n))\)发现\(Y^{'}_{n}\)中值得信赖的区域。半监督损失通过公式8获得，其中\(I(.)\)是指示函数，\(T_{semi}\)和\(\lambda_{th}\)是超参数。 \[ L_{MT}=\left\{ \begin{array}{rcl} -\sum^{H\times W}_{i=1}\sum^{C}_{c=1}I(D_2(S(X_n))^{(i,c)}>T_{semi})Y^{'}_{n}\log{(S(X_n)^{(i,c)})},& D_1(S(X_n))>\lambda_{th} \\ 0,& otherwise \end{array} \right. \tag{8}\]

EXPERIMENTAL RESULTS AND ANALYSIS

为了评估该方法的效果，我们在一个语义分割基准上进行了大量的实验:PASCAL VOC 2012 [13]我们实现了整个模型在一个单一的GPU k40与12G内存。为了训练分割网络，我们采用了初始学习率为\(2.5\times 10^{-4}\)的SGD优化器，学习率服从幂为0.9的多项式衰减，动量为0.9，权值衰减为\(10^{-4}\)[1]。对于鉴别器网络，我们使用了Adam优化器，学习速率为\(10^{-4}\)。批量大小为10。用标准平均相交-过并集(mIoU)来度量性能。

PASCAL VOC 2012 PASCAL VOC 2012包含了20个常见的对象类别和背景。我们的模型是在来自分割边界数据集(SBD)[14]的附加注释图像上训练的，其中包含10582张训练图像和1449张验证图像。表1显示了评估结果2012年帕斯卡VOC数据集和语义分割与其他方法比较基线DeepLab-v2没有多尺度融合和基线+𝐷([1]提出的方法。我们从训练数据集中随机抽取1/8、1/4、1/2的图像作为标记数据，其余的图像作为未标记数据来创建半监督条件[1]。从表1可以看出，我们的方法带来的整体改进是0.9%到2.6%。

为了验证自注意层、跳跃层和双鉴别器的效果，我们还进行了消融研究，标记数据量为1/8。如表2所示，“+”表示我们在前一个模型的基础上添加了新的操作。实验结果表明，所有这些设计都有助于提高系统的性能。图3显示了我们的方法生成的一些可视化分割结果。

CONCLUSION

本文提出了一种用于半监督语义分割的对抗性学习方法。该双重鉴别器不仅能对标记数据进行高阶一致性检验，而且还能生成置信图作为对未标记数据的辅助监控信号。在PASCAL VOC 2012数据集上的一系列实验证明了我们方法的有效性。然而，由于实验时间的不足，实验的精度还存在很大的提高空间。在未来，我们将使用更复杂的数据集来验证我们的方法，并探索可能进一步提高准确性的因素。
